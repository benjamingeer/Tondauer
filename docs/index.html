Tondauer

This web site is for experiments with digital editions of music.

Felix Mendelssohn, Prelude Op. 104a, No. 2 (MWV U 123)

These draft editions are intended to illustrate possible functionality. Source code is at https://github.com/dasch-swiss/Tondauer/.

  • Editions encoded and rendered with LilyPond, in PDF format, for printing or for use with a tablet:
    • Critical edition
    • Performance edition
    • Performance edition (variant)
  • Interactive critical edition (a very simple mockup without much functionality) encoded as MEI and rendered as SVG with Verovio
